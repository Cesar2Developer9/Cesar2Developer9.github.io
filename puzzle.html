Puzzle

Problema

El 8-puzzle es un problema que consiste en encontrar la secuencia de movimientos a realizar para, partiendo de un tablero aleatorio, llegar hasta el tablero solución

Los movimientos permitidos se obtienen desplazando cualquiera de las fichas adyacentes a la casilla vacía.

Representación

La representación se basa en un diccionario en el cual la clave es el estado que se visita, cuando el estado solución es encontrada en el diccionario el algoritmo se detiene y se muestran los resultados.

Implementación

from collections import deque
import copy
import sys
%time
def Puzzle(inicial,final):
    ####################### 
    estados=0 #Iniciar contador que conocer cuantos estados se visitaron
    visitados={} #Iniciar un diccionario y después guardar en él el estado inicial del puzzle
    visitados[str(inicial)]=inicial
    #arreglos para realizar las combinaciones de movimientos
    incx=[0,0,1,-1]
    incy=[1,-1,0,0] 
    width=len(inicial[0])
    height=(len(inicial)-2)
    ######################## 
    #Crear nueva cola y agregar el estado inicial el cual contiene el puzzle
    # y la posición del vacio el cual es representado con 0
    cola=deque()
    cola.append(inicial) 
    tmp=None
    while cola:
        #Extraer elementos de la cola mientras no se encuentre vacía
        tmp=None
        tmp=copy.deepcopy(cola.pop())
        estados=estados+1
        if (tmp==final):
            #Si el primer estado que visitamos ya es solución ejecuta retur para salir del ciclo
            print (tmp)
            print ('Hecho')
            return
        xbefore=tmp['x']#coordenadas del 0 del elemento extraido de la cola
        ybefore=tmp['y']  
        for i in range(4):        
            nx=xbefore+incx[i]#nuevas coordenadas respecto a los arreglos
            ny=ybefore+incy[i]
            #validaciones para no entrar a coordenadas no definidas
            if(nx>=0 and nx<height and ny>=0 and ny<width):
                ###############################
                #Se intercambia el 0 con el número que se encuentra
                #en la nueva posición a un estado temporal
                state=None
                state=copy.deepcopy(tmp)
                movtmp=state[nx][ny]
                state[nx][ny]=0
                state[xbefore][ybefore]=movtmp
                state['x']=nx
                state['y']=ny
                ###############################
                #Si el estano temporal no está en visitados
                #se agrega
                if str(state) not in visitados:
                    visitados[str(state)]=state
                    cola.append(state) 
                    #Si el nuevo estado es la solución entonces retorna
                    if (state==final):
                        print(state)
                        print ('Hecho')
                        sys.getsizeof(cola)
                        #Imprime contador de estados visitados
                        print(str(estados)+' Visitados')
                        return

Ejemplos

El valor de P es el estado inicial que se necesita resolver para llegar a un estado PF definiendo en 'x' y 'y' la posición del vacío representada por 0.

P={}
P[0]=[1,3,8]
P[1]=[0,6,2]
P[2]=[5,4,7]
P['x']=1
P['y']=0

PF={}
PF[0]=[1,2,3]
PF[1]=[4,5,6]
PF[2]=[7,8,0]
PF['x']=2
PF['y']=2
